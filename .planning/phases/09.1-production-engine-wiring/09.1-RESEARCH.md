### §1 Anthropic Python SDK (anthropic==0.75.0) — Production Patterns

**Verified environment:** `python -c "import anthropic; print(anthropic.__version__)"` → `0.75.0` (2026-04-20 실측).

**Canonical call pattern for producer:**
```python
from anthropic import Anthropic, APIStatusError, RateLimitError, APITimeoutError
client = Anthropic()  # reads $ANTHROPIC_API_KEY automatically

response = client.messages.create(
    model="claude-sonnet-4-6-20260301",   # producer
    max_tokens=4096,                        # JSON 은 보통 ≤ 2K tokens
    system=producer_system_prompt,          # AGENT.md body (frontmatter 제거 후)
    messages=[
        {"role": "user", "content": json.dumps(inputs, ensure_ascii=False)},
    ],
    temperature=0.3,                        # 결정론적 경향
)
text = response.content[0].text            # TextBlock.text
parsed = json.loads(text)                   # ValidationError propagate
```

**Key facts (확인 완료):**
- `client.messages.create` 시그니처 파라미터: `max_tokens, messages, model, metadata, service_tier, stop_sequences, stream, system, temperature, thinking, tool_choice, tools, top_k, top_p, extra_headers` — **`response_format` 파라미터 없음** (OpenAI 와 달리 native JSON mode 없음. 3 방법: (a) system prompt 에 명시 + parse 재시도, (b) tool_use 사용 (9.1 scope 외), (c) prefilled assistant `{`).
- `response.content` 는 `list[ContentBlock]`. 첫 block 이 `TextBlock` 일 때 `.text`. 순회 안전: `"".join(b.text for b in response.content if hasattr(b, "text"))`.
- 에러 분류: `RateLimitError` (429) / `APITimeoutError` / `APIStatusError` (500+). CircuitBreaker 는 `(RateLimitError, APITimeoutError, APIStatusError)` 3종만 failure 로 count (나머지 ValueError/JSONDecodeError 는 즉시 re-raise).
- `service_tier="standard_async"` 로 batch 유사 대기열 가능하나 Phase 9.1 에서는 미사용 (Phase 10 로 연기).
- 모델 ID 2026-04 기준: Sonnet 4.6 = `claude-sonnet-4-6-20260301`, Opus 4.6 = `claude-opus-4-6-20260320` (STACK.md 기준; 실제 호출 전 `client.models.list()` 검증 권장하지만 9.1 에서는 하드코딩 허용).

**JSON enforcement triple defense:**
1. system prompt 끝에 RoPE 기반 재배치: `"CRITICAL: Respond with a single valid JSON object. No markdown code fences, no preamble, no commentary. If you cannot comply, return {\"error\": \"reason\"}."`
2. prefill assistant message: `messages=[{"role":"user",...}, {"role":"assistant", "content":"{"}]` — 모델이 `{` 로 이어쓰게 강제 (Anthropic 공식 기법). 이 경우 파싱 시 `"{" + text` 로 재결합.
3. client-side strict `json.loads`. Fail → 1회 재시도 (더 엄격한 system prompt) → Fail → raise (Hook 3종 silent-except 차단).

**Rate limit / retry:**
- anthropic SDK 내장 `max_retries=2` 기본값. CircuitBreaker 밖에서는 `Anthropic(max_retries=0)` 로 무력화 후 CircuitBreaker 에 일원화 권장.
- Sonnet 4.6 input tier 기준 RPM / TPM 한도는 org 별 상이. 9.1 는 smoke test 1-2회 + 테스트만이므로 한도 위험 0.

**Tool selection:**
- Producer (대부분 gate) = Sonnet 4.6, `temperature=0.3`, `max_tokens=4096`.
- Supervisor (Verdict 판정) = Opus 4.6, `temperature=0.0`, `max_tokens=512` (verdict + 짧은 근거).

**Adapter pattern (shorts_studio 신규):**
```python
# scripts/orchestrator/api/nanobanana.py
class NanoBananaAdapter:
    DEFAULT_MODEL = "nano-banana-pro-preview"
    DEFAULT_OUTPUT_DIR = Path("outputs/nanobanana")

    def __init__(self, api_key: str | None = None,
                 circuit_breaker: "CircuitBreaker | None" = None,
                 output_dir: Path | None = None) -> None:
        resolved = api_key or os.environ.get("GOOGLE_API_KEY") or os.environ.get("GEMINI_API_KEY")
        if not resolved:
            raise ValueError("NanoBananaAdapter: GOOGLE_API_KEY 미설정 — 대표님 .env 확인 필요")
        self._api_key = resolved
        self.circuit_breaker = circuit_breaker
        self.output_dir = output_dir or self.DEFAULT_OUTPUT_DIR

    def generate_scene(self, prompt: str, output_path: Path | None = None,
                       model: str | None = None) -> Path:
        # CircuitBreaker 는 orchestrator level 에서 wrap. 여기선 raw call.
        from google import genai
        from google.genai import types

        self.output_dir.mkdir(parents=True, exist_ok=True)
        out = output_path or (self.output_dir / f"scene_{int(time.time())}.png")

        client = genai.Client(api_key=self._api_key)
        try:
            response = client.models.generate_content(
                model=model or self.DEFAULT_MODEL,
                contents=prompt,
                config=types.GenerateContentConfig(response_modalities=["TEXT", "IMAGE"]),
            )
            self._save_first_image(response, out)
            return out
        except Exception as e:
            err = str(e).upper()
            if any(k in err for k in ("SAFETY", "BLOCK", "PROHIBITED")):
                softer = self._soften(prompt)
                response2 = client.models.generate_content(...)
                self._save_first_image(response2, out)
                return out
            raise
```

### §3 FFmpeg Ken-Burns zoompan — local subprocess

**Verified:** `which ffmpeg` → `C:\...\ffmpeg-8.0.1-full_build\bin\ffmpeg`, `ffmpeg -version` → `ffmpeg version 8.0.1-full_build-www.gyan.dev`. `shutil.which("ffmpeg")` 로 precondition 검사 가능.

**Canonical command (Bannerbear + Creatomate 가이드 확인):**
```bash
ffmpeg -y -loop 1 -i input.jpg \
  -vf "scale=4000:-1,zoompan=z='min(zoom+0.0015,1.5)':x=iw/2-(iw/zoom/2):y=ih/2-(ih/zoom/2):d=150:s=720x1280:fps=30" \
  -c:v libx264 -pix_fmt yuv420p -t 5 output.mp4
```

**Key facts:**
- `-loop 1` 필수 (정지 이미지 입력).
- `scale=4000:-1` 먼저 (zoompan 이 low-res 입력에서 jittery). `-1` 은 aspect 유지.
- `zoom` 표현식: `zoom+0.0015` per frame → 5초/150frames 동안 대략 1.0 → 1.225 (22.5% zoom). 필요 시 `min(zoom+0.0015,1.5)` 로 상한.
- `x=iw/2-(iw/zoom/2)` / `y=ih/2-(ih/zoom/2)` = center-anchored zoom.
- `d=150` = frames (duration_frames = duration_sec × fps), `s=720x1280` = output (9:16 Shorts), `fps=30`.
- `-t 5` = output duration seconds (zoompan `d` 와 중복이지만 안전망).
- `-c:v libx264 -pix_fmt yuv420p` → H.264/YUV420 (YouTube Shorts 호환).
- Pan 추가 (좌→우): `x='(iw-iw/zoom)*t/5'` 로 시간 기반 shift.

**Python subprocess pattern:**
```python
import shutil, subprocess
from pathlib import Path

class KenBurnsUnavailable(RuntimeError):
    pass

class KenBurnsLocalAdapter:
    def __init__(self, circuit_breaker=None, output_dir=None):
        if shutil.which("ffmpeg") is None:
            raise KenBurnsUnavailable("ffmpeg 바이너리를 찾을 수 없습니다 — 대표님 PATH 확인 필요")
        self.circuit_breaker = circuit_breaker
        self.output_dir = output_dir or Path("outputs/ken_burns")

    def generate_clip(self, image: Path, duration_seconds: int = 5,
                      resolution: tuple[int, int] = (720, 1280),
                      fps: int = 30, zoom_rate: float = 0.0015) -> Path:
        self.output_dir.mkdir(parents=True, exist_ok=True)
        out = self.output_dir / f"kb_{int(time.time())}.mp4"
        frames = duration_seconds * fps
        w, h = resolution
        vf = (f"scale=4000:-1,zoompan=z='min(zoom+{zoom_rate},1.5)':"
              f"x=iw/2-(iw/zoom/2):y=ih/2-(ih/zoom/2):d={frames}:"
              f"s={w}x{h}:fps={fps}")
        cmd = [
            "ffmpeg", "-y", "-loop", "1", "-i", str(image),
            "-vf", vf, "-c:v", "libx264", "-pix_fmt", "yuv420p",
            "-t", str(duration_seconds), str(out),
        ]
        result = subprocess.run(cmd, capture_output=True, text=True, timeout=120)
        if result.returncode != 0:
            raise RuntimeError(f"ffmpeg 실패 (rc={result.returncode}): {result.stderr[-500:]}")
        return out
```

**Pitfall:** `subprocess.run(..., shell=False)` 유지 (Hook 차단 + injection 방지). stdout capture 필수 (디버깅용). Windows 환경에서 path 는 forward slash OK (`ffmpeg` 정상 파싱).

### §4 pydantic v2 CharacterEntry model

**Canonical schema:**
```python
# scripts/orchestrator/character_registry.py
from pathlib import Path
from typing import Annotated
import json
from pydantic import BaseModel, Field, field_validator

class CharacterEntry(BaseModel):
    model_config = {"extra": "forbid"}   # schema drift 차단

    name: str = Field(min_length=1, max_length=64)
    ref_path: Path
    description: str = Field(default="", max_length=500)
    tags: list[str] = Field(default_factory=list)

    @field_validator("ref_path")
    @classmethod
    def _validate_ref_exists(cls, v: Path) -> Path:
        v = Path(v) if not isinstance(v, Path) else v
        # Path 가 relative 일 수 있으므로 존재 체크는 Registry 레벨로 위임
        return v


class CharacterRegistry:
    def __init__(self, registry_path: Path = Path("assets/characters/registry.json")):
        self.registry_path = registry_path
        self.assets_root = registry_path.parent
        self._entries: dict[str, CharacterEntry] = {}

    def load(self) -> "CharacterRegistry":
        if not self.registry_path.exists():
            raise FileNotFoundError(
                f"캐릭터 레지스트리 파일 없음: {self.registry_path} — "
                f"`assets/characters/registry.json` 을 먼저 생성하세요"
            )
        raw = json.loads(self.registry_path.read_text(encoding="utf-8"))
        for entry_dict in raw.get("characters", []):
            entry = CharacterEntry.model_validate(entry_dict)
            resolved = self.assets_root / entry.ref_path if not entry.ref_path.is_absolute() else entry.ref_path
            if not resolved.exists():
                raise FileNotFoundError(
                    f"캐릭터 '{entry.name}' 레퍼런스 파일 없음: {resolved}"
                )
            self._entries[entry.name] = entry
        return self

    def get(self, name: str) -> CharacterEntry:
        if name not in self._entries:
            raise KeyError(f"캐릭터 '{name}' 을 레지스트리에서 찾을 수 없음 — list_all()={list(self._entries)}")
        return self._entries[name]

    def get_reference_path(self, name: str) -> Path:
        entry = self.get(name)
        return self.assets_root / entry.ref_path if not entry.ref_path.is_absolute() else entry.ref_path

    def list_all(self) -> list[str]:
        return list(self._entries.keys())
```

import httpx, os

def discover_korean_default_voice(api_key: str | None = None) -> str:
    """대표님 ElevenLabs 계정에서 한국어 기본 voice_id 검색.
    우선순위: labels.language == 'ko' → labels.accent == 'korean' → category == 'premade'

    key = api_key or os.environ.get("ELEVENLABS_API_KEY") or os.environ.get("ELEVEN_API_KEY")
    if not key:
        raise ValueError("ElevenLabs API 키 미설정")
    resp = httpx.get(
        "https://api.elevenlabs.io/v1/voices",
        headers={"xi-api-key": key, "accept": "application/json"},
        timeout=30,
    )
    resp.raise_for_status()
    voices = resp.json().get("voices", [])
    # Tier 1: labels.language == 'ko'
    for v in voices:
        if v.get("labels", {}).get("language") == "ko":
            return v["voice_id"]
    # Tier 2: category premade 첫 항목
    for v in voices:
        if v.get("category") == "premade":
            return v["voice_id"]
    # Tier 3: 첫 유효
    if voices:
        return voices[0]["voice_id"]
    raise RuntimeError("대표님 ElevenLabs 계정에 사용 가능한 voice 가 없습니다")

"""Phase 9.1 Claude Agent SDK wiring — production invokers.

Replaces shorts_pipeline._default_producer_invoker / _default_supervisor_invoker
/ _default_asset_sourcer (which currently raise NotImplementedError).
"""
from __future__ import annotations

import json
import logging
from pathlib import Path
from typing import Callable

from anthropic import Anthropic, APIStatusError, APITimeoutError, RateLimitError

from .circuit_breaker import CircuitBreaker
from .gates import GateName
from .gate_guard import Verdict
from .api.nanobanana import NanoBananaAdapter
from .character_registry import CharacterRegistry

logger = logging.getLogger(__name__)

PRODUCER_MODEL = "claude-sonnet-4-6-20260301"
SUPERVISOR_MODEL = "claude-opus-4-6-20260320"

_JSON_INSTRUCTION = (
    "\n\nCRITICAL: 반드시 단일 valid JSON 객체로만 응답하세요. "
    "markdown code fence, preamble, commentary 전부 금지. "
    "준수 불가 시 {\"error\": \"<사유>\"} 로 응답."
)


def load_agent_system_prompt(agent_dir: Path) -> tuple[str, dict]:
    """Parse AGENT.md frontmatter and body. Body becomes Claude system prompt."""
    content = (agent_dir / "AGENT.md").read_text(encoding="utf-8")
    if content.startswith("---\n"):
        parts = content.split("---\n", 2)
        if len(parts) >= 3:
            import yaml
            frontmatter = yaml.safe_load(parts[1]) or {}
            body = parts[2].strip()
            return body, frontmatter
    return content.strip(), {}


class ClaudeAgentProducerInvoker:
    """Callable invoker matching (agent_name, gate_name, inputs) -> dict."""

    def __init__(self, agent_dir_root: Path, circuit_breaker: CircuitBreaker | None = None,
                 client: Anthropic | None = None):
        self.agent_dir_root = agent_dir_root
        self.circuit_breaker = circuit_breaker
        self.client = client or Anthropic(max_retries=0)  # CircuitBreaker 일원화

    def __call__(self, agent_name: str, gate: str, inputs: dict) -> dict:
        agent_dir = self.agent_dir_root / agent_name
        if not agent_dir.exists():
            raise FileNotFoundError(f"Agent 디렉토리 없음: {agent_dir} — Phase 4 AGENT.md 확인")

        system, frontmatter = load_agent_system_prompt(agent_dir)
        system_full = system + _JSON_INSTRUCTION

        user_payload = json.dumps({"gate": gate, "inputs": inputs}, ensure_ascii=False)

        def _call():
            return self.client.messages.create(
                model=PRODUCER_MODEL,
                max_tokens=4096,
                system=system_full,
                messages=[
                    {"role": "user", "content": user_payload},
                    {"role": "assistant", "content": "{"},   # JSON prefill
                ],
                temperature=0.3,
            )

        try:
            resp = self.circuit_breaker.call(_call) if self.circuit_breaker else _call()
        except (RateLimitError, APIStatusError, APITimeoutError) as e:
            logger.error("[invoker] producer API 실패 (대표님): %s", e)
            raise

        text = "{" + "".join(b.text for b in resp.content if hasattr(b, "text"))
        try:
            return json.loads(text)
        except json.JSONDecodeError as e:
            logger.error("[invoker] JSON parse 실패: %s — first 200 chars: %s", e, text[:200])
            raise RuntimeError(f"Producer '{agent_name}' JSON 미준수: {e}") from e


class ClaudeAgentSupervisorInvoker:
    """Callable invoker matching (GateName, output) -> Verdict."""

    def __init__(self, agent_dir: Path, circuit_breaker: CircuitBreaker | None = None,
                 client: Anthropic | None = None):
        self.agent_dir = agent_dir
        self.circuit_breaker = circuit_breaker
        self.client = client or Anthropic(max_retries=0)
        self._system, _ = load_agent_system_prompt(agent_dir)
        self._system_full = self._system + _JSON_INSTRUCTION

    def __call__(self, gate: GateName, output: dict) -> Verdict:
        user_payload = json.dumps({"gate": gate.name, "producer_output": output}, ensure_ascii=False)

        def _call():
            return self.client.messages.create(
                model=SUPERVISOR_MODEL,
                max_tokens=512,
                system=self._system_full,
                messages=[
                    {"role": "user", "content": user_payload},
                    {"role": "assistant", "content": "{"},
                ],
                temperature=0.0,
            )

        resp = self.circuit_breaker.call(_call) if self.circuit_breaker else _call()
        text = "{" + "".join(b.text for b in resp.content if hasattr(b, "text"))
        parsed = json.loads(text)

        verdict_str = (parsed.get("verdict") or "").upper()
        try:
            return Verdict[verdict_str]   # PASS / FAIL / RETRY enum
        except KeyError:
            raise RuntimeError(f"Supervisor verdict 파싱 실패: {verdict_str!r}")


def make_default_producer_invoker(agent_dir_root: Path, circuit_breaker: CircuitBreaker | None = None):

## Risks / Pitfalls

### Pitfall 1: Anthropic JSON prefill 파싱 누락
**What goes wrong:** `assistant` 메시지에 `{` 를 prefill 로 넣으면 response.content 의 첫 block 은 `{` 없이 시작 — 클라이언트에서 다시 `"{" + text` 로 재결합하지 않으면 JSONDecodeError.
**How to avoid:** invokers.py 에서 `text = "{" + "".join(b.text for b in resp.content if hasattr(b, "text"))` 로 prefill 복원. 테스트에서 이 동작 명시적으로 검증.

### Pitfall 2: AGENT.md 크기 폭발
**What goes wrong:** scripter AGENT.md 는 211줄. 전체 Producer 14 × 평균 200줄 = 시스템 프롬프트 ~2800 tokens. Sonnet 4.6 는 수용 가능하나 매 호출 반복 → 월 비용 증가.
**How to avoid:** Phase 9.1 에서는 그대로 주입 (prompt caching 미적용). Phase 10 에서 `anthropic` prompt caching (`cache_control: {type: "ephemeral"}`) 로 system 블록 캐싱하여 비용 90% 절감.

### Pitfall 3: zoompan jitter on low-resolution input
**What goes wrong:** 원본 이미지가 720x1280 이면 zoompan 이 4000px 로 upscale 하지 않고 곧바로 zoom 하면 프레임 간 지터 발생.
**How to avoid:** filter_complex 순서 `scale=4000:-1,zoompan=...` 고정. 테스트 `test_cmd_construction` 에서 `scale=4000` 포함 여부 assert.

### Pitfall 4: Runway model 교체 시 ratio 자동 변경으로 silent 충돌
**What goes wrong:** 테스트 코드가 `RunwayI2VAdapter(model="gen4.5", ratio="768:1280")` 생성 시도 → 유효 ratio 목록 `["720:1280"]` 미포함 → ValueError. 9.1 에서 smoke test 는 `gen3a_turbo` 고정이므로 영향 없으나 Phase 10 migration 시 문제.
**How to avoid:** `VALID_RATIOS_BY_MODEL` 가 **source of truth** 임을 docstring 명시. Phase 10 새 모델 추가 시 이 dict 를 먼저 갱신하라는 note.

### Pitfall 5: CharacterRegistry ref_path 상대경로 vs 절대경로
**What goes wrong:** registry.json 에 `"ref_path": "channel_profile_ref.png"` (상대) 로 저장. 하지만 pipeline 이 `cwd` 가 다른 위치에서 실행되면 파일 resolve 실패.
**How to avoid:** `CharacterRegistry.__init__` 에서 `self.assets_root = registry_path.parent` 를 기록하고 `get_reference_path` 가 `self.assets_root / entry.ref_path` 로 조합 (절대경로는 그대로 유지).

### Pitfall 6: voice_discovery API 호출이 adapter 생성 시마다 반복
**What goes wrong:** `ElevenLabsAdapter()` 생성 시마다 `GET /v1/voices` 호출 → 대표님 API quota 낭비 + 느림.
**How to avoid:** 모듈 레벨 `_KOREAN_FALLBACK_VOICE_ID` 전역 캐시. env `ELEVENLABS_DEFAULT_VOICE_ID` 우선 (권장: 대표님 `.env` 에 추가 후 discovery 는 1회만).

### Pitfall 7: Hook 3종 silent-except 재발 — safety fallback 에서
**What goes wrong:** Nano Banana safety fallback 의 `except Exception as e: if "SAFETY" in str(e).upper(): retry; else: raise` — 일부 ValueError / APIStatusError 를 숨기는 셈이 되면 Hook 차단.
**How to avoid:** except 블록 안에서 **항상 raise 또는 로그 후 raise** 한다. "softer prompt 재시도" 도 성공 시 반환 / 실패 시 재raise 명시. `except Exception as e: logger.warning(...); if safety: retry_ok = _retry(...); if retry_ok: return out; raise`.

### Pitfall 8: shorts_pipeline.py 줄수 초과
**What goes wrong:** 2개 신규 adapter slot (nanobanana, ken_burns) 추가 + 4 CircuitBreaker 추가 + default invoker factory 호출 교체 시 ~30줄 추가 → 800 초과 위험.
**How to avoid:** 모든 factory 호출은 `invokers.py` 로 이동 (이미 §1 에서 반영). adapter constructor 도 invokers.py 의 `make_default_adapters()` helper 로 묶기. `shorts_pipeline.py` 순변경 ≤ 20줄 목표. 테스트 `test_pipeline_line_count.py` 가 ≤800 assert.

### Pitfall 9: 대표님 계정에 ElevenLabs 한국어 voice 가 없는 경우
**What goes wrong:** `discover_korean_default_voice` 가 `labels.language == 'ko'` voice 없을 경우 `premade` fallback 으로 영어 voice 반환. 한국어 대본에 영어 voice 합성 → 대참사.
**How to avoid:** 대표님 계정 2026-04-20 실측 시점에 Korean voice 부재 → **Wave 1 plan 에 수동 검증 태스크 추가**: smoke test 이전에 대표님에게 "ElevenLabs 계정에서 한국어 voice 1개를 즐겨찾기 하거나 `ELEVENLABS_DEFAULT_VOICE_ID` 환경변수로 명시" 요청. fallback 도 반드시 Korean 체크; 없으면 `RuntimeError("한국어 voice 미발견 — 대표님 ElevenLabs 계정에서 Korean voice 추가 또는 ELEVENLABS_DEFAULT_VOICE_ID env 설정 필요")`.

### Pitfall 10: Claude Agent SDK 호출이 Phase 7 regression 을 깨뜨리는 경우
**What goes wrong:** Phase 7 E2E mock test 는 `MagicMock` invoker 주입 기반 — 새 `ClaudeAgentProducerInvoker` 가 default 되면 테스트 환경에서 실 API 호출 시도 → flaky.
**How to avoid:** 기존 Phase 7 테스트는 이미 `MagicMock` 을 명시적으로 주입 중. 그대로 유지. 새 default 는 **production 경로** 에만 적용. `tests/phase091/` 도 mock invoker 주입 유지. 회귀 시 Wave 0 acceptance 에 `pytest tests/phase07/ -q` 포함.
