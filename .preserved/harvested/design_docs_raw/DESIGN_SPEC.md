┌─────────────────────┐ 0px

│                     │

│  풀스크린 영상        │ 1080x1920 전체

## 2. 제목 — 픽셀 사양

### 위치
- Y 시작: **150-220px** (화면 상단 8-11%)
- 제목 영역 높이: **200-400px** (전체의 10-20%)

### 폰트 크기 (1080x1920 기준)
- **1줄 (상황/맥락)**: 60-76px
- **2줄 (핵심 키워드)**: 72-103px (1줄보다 크거나 같음)
- 우리 기존: 48/64px → **너무 작음. 최소 70/90px 이상**

### 텍스트 스타일
- **폰트**: ExtraBold/Black (가장 굵은 웨이트)
- **외곽선**: 검정 4-6px (1080 기준)
- **그림자**: 드롭 섀도우 3-4px, 45도 방향, 검정 80% 불투명
- **줄간격**: 폰트 크기의 120-140%

### 강조 색상 (정밀 측정값)
| 채널 유형 | 색상 | Hex 범위 |
|----------|------|----------|
| 노랑 강조 | 골드~옐로우 | #FFD000 ~ #FFDB04 |
| 빨강 강조 | 선명한 레드 | #FF1005 ~ #FF4A4D |
| 주황 강조 | 오렌지 | #FF9700 ~ #FFA24D |

### 강조 적용 방식
- 2줄 제목 중 **핵심 키워드만** 색상 변경
- 나머지는 흰색(#FFFFFF)
- 예: "직원들 살 떨리는" (흰) + "재벌 사모님의 패션 오더" (**빨강**)

## 3. 자막 — 픽셀 사양

### 위치
- **Y 위치: 1200-1400px** (화면의 62-73%)
- **영상 위에 떠있음** (검정바 안이 아님!)
- 화면 하단 20-38% 영역

### 폰트 크기
- **메인 자막: 45-58px** (1080 기준)
- 우리 기존: 32-40px → 약간 키워야 함

### 스타일
- **색상**: 흰색 (#FFFFFF)
- **외곽선**: 검정 3-4px
- **그림자**: 검정 2px 드롭 섀도우
- **정렬**: 가운데 정렬
- **최대 너비**: 화면 폭의 85% (918px)

### 자막 표시 방식
- 5-10자 짧은 구 단위
- 내레이션 타이밍과 정확히 싱크
- 한 번에 1줄만 표시

## 4. 하단 채널 정보

### 위치
- Y: **1750-1870px** (화면 하단 5-8%)
- 왼쪽 또는 가운데 정렬

### 내용
- 채널 로고/아이콘 (있는 경우)
- 채널명 텍스트
- 해시태그 (#클라이맥스, #게임잡지 등)
- 출처 표기 (©, 원본 출처)

### 스타일
- 폰트: 18-24px (작게)
- 색상: 흰색 또는 회색, 반투명
- 배경: 없음 또는 반투명 검정

## 6. FFmpeg 구현 사양

### 제목 오버레이 (ASS)
```
Style: TitleLine1,Pretendard,70,&H00FFFFFF&,&H00FFFFFF&,&H00000000&,&H50000000&,-1,0,0,0,100,100,0,0,1,5,3,8,20,20,{MarginV_line1},1
Style: TitleLine2,Pretendard,90,&H00FFFFFF&,&H00FFFFFF&,&H00000000&,&H50000000&,-1,0,0,0,100,100,0,0,1,6,4,8,20,20,{MarginV_line2},1
```
- Alignment=8 (상단 가운데)
- MarginV_line1 = 50 (상단에서 50px)
- MarginV_line2 = 130 (1줄 아래)
- Outline=5~6, Shadow=3~4

### 자막 오버레이 (SRT → ASS 변환 또는 force_style)
```
FontSize=50,Bold=1,PrimaryColour=&H00FFFFFF&,OutlineColour=&H00000000&,Outline=4,Shadow=2,Alignment=2,MarginV=500
```
- MarginV=500 → 화면 하단에서 500px 위 = Y≈1420px (74% 위치)
- Alignment=2 (하단 가운데)

### 영상 = 풀스크린
```
ffmpeg -i input.mp4 -vf "scale=1080:1920:force_original_aspect_ratio=increase,crop=1080:1920" output.mp4
```
영상을 강제로 1080x1920에 맞춤. 검정바 없음.
