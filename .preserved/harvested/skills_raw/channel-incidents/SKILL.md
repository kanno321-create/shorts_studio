## 🎬 장르 정의: Cinematic True Crime Documentary

이 채널은 단순 정보 전달이나 기사 형식이 **절대** 아니다.
**시네마틱 트루크라임 다큐멘터리** 스타일로 서사를 구성한다.

네 가지 핵심 무기:

### 무기 1: 영화적 콜드 오프닝 (Cinematic Cold Open)

**육하원칙으로 시작하지 마라.** 사건의 가장 극적·기괴·긴장감 넘치는 특정 순간을 **소설처럼 묘사하며 시각적으로** 시작한다.

가능하면 **아무것도 모르는 무고한 인물**(어린아이, 이웃, 평범한 목격자)의 시선에서 현장의 기이함이나 불안감을 **감각적으로** 묘사한다.

BAD 예시: "1999년 나고야에서 한 여성이 살해당했습니다."
GOOD 예시: "1999년 나고야. 감을 나눠주러 2011호 초인종을 누르던 집주인. 문고리가 그대로 돌아갔고, 미세한 TV 소리 사이로 복도 끝에 누군가의 다리가 보였습니다. 그 아래, 엄청난 양의 피가 웅덩이처럼 고여 있었죠."

**감각 묘사의 예**:
- 시각: 피가 고인 복도, 텅 빈 차 안, 열려있는 문
- 촉각: 발목에서 뻗어오는 불안감, 문고리의 감촉
- 청각: 미세한 TV 소리, 멀리서 부르는 소리
- 냄새/미각: 의심스러운 고요함

### 무기 2: 탐정 시점의 능동적 화자 (First-Person Detective)

전지적 작가 시점이 아니라 **이 사건을 직접 조사하고 파헤치는 '탐정/조사관(나)'의 1인칭 화법**을 쓴다.

독자에게 **직접 말을 걸며 추리의 과정으로 안내**한다:

```
"제가 이 기록을 보았을 때 소름이 돋았습니다"
"여기서 우리는 한 가지 의문을 품게 됩니다"
"이때 결정적인 단서가 등장합니다"
"저는 자료를 확인했을 때 헛웃음을 숨길 수 없었습니다"
"이때쯤이면 한 가지 의문이 생깁니다"
```

탐정은 **화자이자 안내자**다. 시청자와 함께 퍼즐을 맞춰나간다.

### 무기 3: 미스터리 빌드업 + 플롯 트위스트 (Misdirection → Reveal)

**처음부터 범인이나 진실을 말하지 마라.**

1. **오프닝**: 단순 사고·자살·타인의 소행처럼 보이게 **미스디렉션**
2. **중반**: 현장의 기묘한 단서들을 **하나씩 나열** (퍼즐 조각)
   - 부검 결과의 모순
   - 뜬금없는 증거물 (특정 지역 유산균 음료, 바다시신의 민물)
   - 시간의 어긋남, 장소의 부자연스러움
3. **중후반**: **가장 충격적인 진실** 터뜨림
   - 범인이 26년 전 마음을 거절당한 고등학교 동창
   - 보험금을 노린 친아버지
   - 자기 자신을 살인 청부한 14살 소년

**단서는 뜸을 들여라.** 한 번에 쏟지 말고, "그런데...", "그런데 여기서...", "하지만..." 으로 전환하며 긴장감 축적.

### 무기 4: 감정적 여운 + 후일담 (Emotional Aftermath)

**범인을 잡았다고 이야기를 끝내지 마라.** 남겨진 사람들의 비극을 깊이 조명하여 **먹먹한 여운**을 남긴다.

예시:
- 평생 '살인자의 아들'이라는 굴레에 갇혀 문신을 새긴 채 살아가는 소년
- 아내를 잃고 26년간 사건 현장 아파트의 월세를 내며 현장을 보존한 남편의 순애보
- 유가족의 깊은 상처, 눈물겨운 헌신

**마무리 문장의 톤**:
- 인간의 탐욕에 대한 씁쓸함
- 남겨진 자들에 대한 연민
- 해결 이후에도 가시지 않는 무게

## 대본 구조 (4무기 통합)

```
[1] Cinematic Cold Open (5-10초)
    - 무고한 인물의 시점 또는 가장 극적인 순간
    - 감각 묘사로 시청자 몰입
    - "오늘의 기록" 인삿말 삽입

[2] Detective Narrator 등장 (5초)
    - "제가 이 기록을 처음 펼쳤을 때..."
    - 탐정이 조사를 시작하는 선언

[3] Misdirection Phase (20-30초)
    - 사건의 표면적 사실 제시
    - 단순 사건처럼 보이게 유도
    - 첫 번째 이상한 점 슬쩍 제시

[4] Clue Build-up (30-40초)
    - 퍼즐 조각들을 하나씩
    - "그런데 여기서", "하지만" 전환
    - 탐정의 내면 독백으로 서스펜스 축적

[5] Plot Twist / Reveal (15-20초)
    - 가장 충격적인 진실
    - "그리고 저는 깨달았습니다"
    - 시청자의 예상을 뒤집음

[6] Emotional Aftermath (10-15초)
    - 남겨진 사람의 비극
    - 해결 후에도 남은 상처
    - 먹먹한 여운
```

## 서술자: 유능한 탐정 (Detective POV)

이 채널의 나레이션은 **유능한 탐정이 현장을 방문하여 독백하는 시점**으로 작성한다.
3인칭 뉴스 앵커("사건이 발생했습니다")가 아니라, 1인칭 탐정("저는 이 현장을 직접 봤습니다")이다.

### 탐정 시점의 3요소

**1. 현장 관찰** — 탐정이 직접 현장에 있는 듯한 묘사
```
"거실에 종이 인형이 있군요..."
"여기를 보면, 에어컨 실외기에 발자국이 남아 있습니다."
"이 경로는 사전에 관찰하지 않으면 알 수 없는 경로입니다."
```

**2. 증거 분석 → 추론** — 팩트 제시 후 탐정의 해석
```
"이 행동이 의미하는 건 하나입니다."
"이 상처를 보면 얼마나 처절하게 싸웠는지 알 수 있습니다."
"이십오 년간의 수사 방향이 틀렸던 겁니다."
```

**3. 범인 심리 프로파일링** — 증거에서 성격/동기를 추론
```
"극도로 둔감하거나, 극도로 계획적이었음을 보여줍니다."
"충동이 아니라 계획이었습니다."
"이 사소한 행동 하나가 범인의 성격을 말해줍니다."
```

### 정보 출처에 따른 화법 구분

탐정이 직접 볼 수 있는 것과, 물어봐야 아는 것을 구분하라.

| 정보 유형 | 화법 | 예시 |
|----------|------|------|
| **현장 관찰** (직접 봄) | 단정형 | "거실에 종이 인형이 있군요." |
| **주변 증언** (물어봄) | 전달형 | "주변에선 평범하고 화목한 가족이었다고 합니다." |
| **수사 자료** (기록) | 사실형 | "DNA 분석 결과, 부계는 동아시아계입니다." |
| **탐정 추론** (판단) | 추론형 | "이 행동은 계획적이었음을 보여줍니다." |

```
BAD:  "평범하고 화목한 가족이었습니다." ← 탐정이 어떻게 아는가?
GOOD: "주변에선 평범하고 화목한 가족이었다고 합니다." ← 출처가 명확
```

### 탐정의 감정 표현 (절제하되 존재)

Part당 **1-2회만**. 남발하면 감정이 약해진다.

```
"수십 년 현장을 봐왔지만, 이건 오래 기억에 남았습니다."
"저를 가장 불안하게 만드는 증거는 따로 있습니다."
"저는 이 사건을 아직 놓지 않았습니다."
```

### 대본 구조 (탐정 시점)

```
Hook: "오늘의 기록." + 사건 핵심 + "저는 이 현장을 직접 봤습니다." (탐정 등장)
Body: 관찰 → 추론 → 다음 증거로 전환 (반복)
CTA: 탐정의 개인적 마무리 (여운형, 구독 직접 언급 금지)
```

## 문장 연결재 (Sentence Connectors) — 로봇적 나열 금지

"A입니다. B입니다. C입니다." 나열은 보고서이지 독백이 아니다.
사람이 혼잣말하듯 이야기할 때는 문장 사이에 **연결 숨결**이 있어야 한다.

### 삽입 빈도: 3-4문장에 1회

매 문장에 붙이면 과하고, 아예 없으면 로봇적이다.

### 연결재 유형

| 유형 | 예시 | 사용 시점 |
|------|------|----------|
| **관찰** | "...군요", "보이는군요", "있군요" | 새로운 것을 발견했을 때 |
| **전환** | "그런데", "흥미로운 건", "한 가지" | 화제를 바꿀 때 |
| **발견** | "이건...", "여기를 보면", "주목할 점이 있습니다" | 증거를 짚을 때 |
| **추론** | "~라는 뜻입니다", "~라는 뜻이죠", "결국 이건..." | 팩트에서 결론 도출 |
| **감탄** | "참...", "아..." | 감정이 배어나올 때 (드물게) |
| **순서** | "먼저", "그리고", "그다음" | 증거를 나열할 때 |

### BAD vs GOOD 예시

```
BAD (로봇):
"거실에 종이 인형이 있습니다. 아버지 미키오가 만들어준 것입니다.
 아내 야스코는 학원을 운영했습니다. 딸 니이나는 여덟 살입니다."

GOOD (탐정 독백):
"거실에 종이 인형이 있군요... 아버지 미키오가 아이들에게 만들어준 것이었습니다.
 아내 야스코는 이 집에서 학원을 운영했습니다.
 그들의 딸 니이나, 여덟 살. 발레를 했습니다."
```

## 탐정 독백의 흐름 (Monologue Flow)

탐정이 현장을 걸으며 독백할 때는 **생각의 흐름**이 있다.
팩트를 나열하는 게 아니라, 하나를 보고 → 떠올리고 → 다음으로 넘어가는 연상 구조.

### Good Flow (연상 구조)
```
"거실에 종이 인형이 있군요..."         ← 현장에서 뭔가 봄
"아버지 미키오가 아이들에게              ← 그걸 보고 배경을 떠올림
 만들어준 것이었습니다."
"주변에선 평범하고 화목한                ← 주변 증언으로 확장
 가족이었다고 합니다."
"아내 야스코, 발레를 사랑하는            ← 가족을 하나씩 소개 (쉼표 나열)
 딸 니이나, 기차를 좋아하는 아들 레이."
"평범하고 화목한 가족..."               ← 여백 (탐정이 잠시 생각에 잠김)
"하지만 누군가가 이 가족을               ← 전환 (탐정의 직감)
 지켜보고 있었습니다."
```

### Bad Flow (나열)
```
"아버지 미키오. 마흔네 살."              ← 신상 나열
"어머니 야스코. 마흔한 살."              ← 또 나열
"딸 니이나. 여덟 살."                   ← 또 나열
"아들 레이. 여섯 살."                   ← 리스트를 읽는 느낌
```

**왜 Bad인가**: 탐정은 현장에서 무언가를 **보고** 이야기를 시작한다. "종이 인형을 보고 → 아버지를 떠올리고 → 가족을 떠올리고 → 슬퍼지고 → 하지만..." 이 연상의 흐름이 없으면 수사 보고서를 읽는 것과 같다.

### 연상 트리거 (씬 시작 시)

| 관찰 대상 | 연상 | 예시 |
|----------|------|------|
| 물건 | 소유자의 성격/생활 | "종이 인형이 있군요" → 다정한 아버지 |
| 장소 | 사건의 배경/시간 | "뒤편에 공원이 있습니다" → 침입 경로 |
| 증거 | 범인의 행동/심리 | "물을 내리지 않았습니다" → 둔감/계획적 |
| 상처 | 격투/저항 | "다리에 상처가 남았습니다" → 필사적 저항 |

## Reference Scripts (골든 샘플 — 대표님 승인 대본)

대본 작성 시 아래 샘플의 톤, 리듬, 흐름, 문장 길이를 참조하라.
이 샘플은 실제 제작되어 대표님이 "올려도 될 정도"라고 승인한 대본이다.

### 샘플 1: 가족 소개 + 배경 (세타가야 Part 1)

> "2000년 겨울, 도쿄 세타가야. 미야자와 미키오는 아이들에게 종이 인형을 만들어주는 다정한 아버지였습니다. 아내 야스코, 발레를 사랑하는 딸 니이나, 기차를 좋아하는 아들 레이. 평범하고 화목한 가족... 하지만 누군가가 이 가족을 지켜보고 있었습니다."

**좋은 이유**: 짧은 세팅(12자) → 긴 디테일(35자) → 쉼표 나열(28자) → 여백(10자) → 반전(20자)

### 샘플 2: 사건 전개 (세타가야 Part 1)

> "12월 30일, 연말의 밤. 밤 열시 삼십팔분, 미키오가 업무 이메일을 확인합니다. 이것이 가족의 마지막 기록입니다... 한 시간 뒤. 누군가 뒷편 공원에서 담을 넘어, 이층 욕실 창문으로 침입합니다."

**좋은 이유**: 시간 구체성(열시 삼십팔분) + "마지막 기록입니다..."로 여백 → 긴장 + 현재형 전환("확인합니다", "침입합니다")

### 샘플 3: 기괴한 행동 묘사 (세타가야 Part 2)

> "범인은 냉장고를 열었습니다. 아이스크림 네 개와 멜론을 꺼내 먹고, 보리차를 마셨습니다. 가족 네 명을 죽인 그 집에서. 이층 소파에 누워 잠까지 잔 것으로 추정됩니다."

**좋은 이유**: 살인 직후 + 일상적 행동(아이스크림/멜론/보리차)의 대비. "가족 네 명을 죽인 그 집에서." — 짧은 펀치가 여기서만 등장 (절제)

### 샘플 4: 증거 분석 (세타가야 Part 2)

> "범인은 전화선도 끊었습니다. 하지만 선을 자르거나 부순 게 아닙니다. 벽에서 조용히 뽑아두었습니다. 이 사소한 행동 하나가, 범인이 극도로 침착했음을 보여줍니다."

**좋은 이유**: "하지만" 전환으로 기대 뒤집기 + 관찰 → 반전 → 추론의 3단계 흐름

### 샘플 사용법

1. 가족/인물 소개 → 샘플 1처럼 쉼표 나열 + "하지만" 전환
2. 사건 전개 → 샘플 2처럼 시간 구체성 + 현재형 전환
3. 기괴한 행동 → 샘플 3처럼 일상 행동 나열 + 대비 펀치
4. 증거 분석 → 샘플 4처럼 관찰 → 반전 → 추론

## Tone

- **형사 보고서 톤 허용**: "극도로 침착했습니다", "잔혹한", "참혹한", "처절하게" — 장르에 맞는 자연스러운 표현
- **AI 클리셰 금지**: "믿으시겠습니까", "놀라운 사실이 하나 더 있습니다", "상상할 수 없는" — 절대 사용 금지
- **능동태 지향**: "발견되었습니다" 대신 "경찰은 발견했습니다", "범인은 냉장고를 엽니다"
- **법의학적 구체성**: "모래가 나왔다" 대신 "에드워즈 공군기지 인근 모하비 사막의 모래"

## 문체 규칙 — 서사체 필수 (FAIL-SCR-001, 세션 40)

탐정 독백은 **서사체** 로 흘러야 한다. **개조식·체언 종결** 은 경찰 보고서·뉴스 자막 톤이라 시네마틱 다큐와 정반대다.

### 절대 금지 — 체언(명사) 종결 단문 연속

대표님 직접 피드백 (세션 40, 2026-04-10):
> "깜빡이는 형광등 아래 소녀가 복도를 달립니다. 17살 팔과다리에 전기화상 — 이런 식의 딱딱하며 AI 스런 대본 말고, 깜빡이는 형광등 아래 팔과다리에 전기화상을 입은 17살 소녀가 복도를 급하게 달립니다 — 이런 식으로 형용사도 좀 넣고, 단어와 단어 사이에 뭔가 연결고리가 있어야 되."

**금지 패턴**: 명사/체언 + 마침표로 끝나는 단문을 **2개 이상 연달아** 배치.

```
❌ BAD — 개조식 나열 (AI 스런 "끊긴" 문체)
"2002년 3월, 기타큐슈. 형광등이 깜빡이는 복도를 맨발의 소녀가 달립니다. 열일곱 살. 팔에 전기 화상. 이 소녀가 입을 열지 않았다면..."

✅ GOOD — 서사체 + 관형절 + 연결 형태소
"2002년 3월, 기타큐슈의 새벽. 깜빡이는 형광등 아래, 팔과 다리에 전기 화상을 입은 열일곱 살 소녀가 맨발로 복도를 급하게 달립니다. 이 소녀가 입을 열지 않았다면..."
```

**핵심 변환 원칙**:

1. **정보 조각은 관형절로 묶어라**
   - "열일곱 살." + "팔에 전기 화상." → "팔과 다리에 전기 화상을 입은 열일곱 살 소녀"
   - 짧은 체언 2개가 등장하면 둘 다 앞 명사를 수식하는 **관형절** 로 변환

2. **연결 형태소로 문장을 이어라**
   - `-고` / `-며` / `-어서` / `-는데` / `-자마자` / `-ㄴ 채` / `-ㄴ 뒤에`
   - "문을 열었습니다. 안에는 피가 고여 있었습니다." → "문을 열자, 안에는 피가 웅덩이처럼 고여 있었습니다."

3. **형용사·부사 허용 (incidents 한정)**
   - `급하게`, `기이하게`, `섬뜩하게`, `처절하게`, `조심스럽게`, `묵묵히`, `차갑게`
   - 메모리 `feedback_channel_adjective_style.md` 에 따라 incidents/politics 는 형사톤 형용사 허용. humor/trend 는 금지.

4. **모든 문장은 서술어로 끝내라**
   - 한국어 술어형 어미: `다/까/죠/네/요/군/러/야/지/며/고/서/랴`
   - 체언 + 마침표는 드문 호흡 장치로만 (한 섹션에 **1회 이하**)

5. **예외 — "1회성 호흡 장치"**:
   긴 서사체 2-3 문장 뒤에 단문 하나로 리듬 전환은 허용. 단, 단문도 **술어 종결** 이어야 한다.
   ```
   "탐정은 피해자의 마지막 메시지를 읽었습니다. 단 한 줄이었습니다."
   ```
   "단 한 줄." (체언 종결) 이 아니라 "단 한 줄이었습니다." (술어 종결).

### 평균 문장 길이 가이드

| 섹션 | 평균 문장 길이 | 비고 |
|------|---------------|------|
| Hook (cold open) | 20-35자 | 서사체 긴 문장 위주 |
| Body (탐정 독백) | 15-25자 | 관찰/추론/심리 분석 섞기 |
| Climax (반전) | 12-20자 | 짧은 호흡 허용 (단, 술어 종결) |
| CTA (여운) | 15-25자 | 긴 여운 문장 |

평균이 **12자 미만** 이면 개조식 나열 의심 → Scripter 가 Step 6 검증에서 경고.

## CTA + 채널 시그니처 캐치프레이즈 — 의무

> **이 규칙은 FAIL-SCR-002 를 방지하기 위한 강제 규칙이다. 위반 시 Scripter wakeup 재호출.**

### 인삿말 (Hook 의무)

**Hook section[0] 의 narration 첫 또는 이른 위치** 에 `"오늘의 기록."` 을 **반드시** 포함한다.

- **배치**: 콜드 오프닝의 감각 묘사 직후, 탐정이 처음 독자에게 말을 거는 지점
- **금지**: body 섹션 중간에 옮기기, 아예 빼기, 변형 ("저의 기록" 같은 변주 금지)

```
✅ GOOD 예시:
"2002년 3월 기타큐슈의 새벽. 깜빡이는 형광등 아래 … 맨발로 복도를 달립니다.
 **오늘의 기록.** 기타큐슈 연쇄 감금 살인. 저는 이 사건을 다시 펼쳤을 때…"

❌ BAD 예시 (세션 40 실제 실수):
Hook: "2002년 3월, 기타큐슈. 형광등이…" (인삿말 없음)
Body[1]: "**오늘의 기록.** 기타큐슈 연쇄 감금 살인…" (인삿말이 body 에 혼입)
```

### 끝맺음말 (CTA 의무)

**마지막 section (type: cta) 의 narration** 에 `"저는 이 사건을 아직 놓지 않았습니다."` 를 **반드시** 포함한다.

- **단편**: 시그니처 끝맺음말 단독으로 종결
- **시리즈 Part N (N < M)**: 시그니처 + 연결형 **둘 다** 사용
  ```
  "저는 이 사건을 아직 놓지 않았습니다. 다음 기록에서, 이어집니다."
  ```
- **시리즈 마지막 편**: 시그니처만
- **구독/좋아요 직접 언급 금지**: 무거운 톤을 깨뜨림

### 왜 의무인가

메모리 `feedback_channel_catchphrase.md` — 대표님이 오래 회의 끝에 결정한 **채널 브랜드 자산**. 매 영상마다 자리가 바뀌거나 빠지면 구독자의 반사 인식 실패 → 브랜드 약화. 심리 리서치 기반 캐치프레이즈이므로 Scripter 의 자유재량 영역이 아니다.

## 🔴 듀오 대화 자연화 규칙 (대표님 2026-04-18 지시)

`incidents` 채널은 **narrator(탐정) + assistant(조수 즌다)** 듀오 포맷. 듀오 대화가 AI티 없이 자연스러워야 한다. 일본어 레퍼런스 `output/mary-celeste-jp/script.json` 패턴 따름.

### 1. 조수 "탐정님" 호명 절대 금지

memory `feedback_assistant_no_detective_honor` (2026-04-18)

- ❌ "**탐정님,** 그토록 많은 사람을 잔혹하게 살해했다면 분명 엄청난 금전적 이득을..."
- ✅ "그토록 많은 사람을 잔혹하게 살해했다면 분명 엄청난 금전적 이득을..."

**이유**: 호명이 교과서 말투 + AI티를 만든다. 자연스러운 대화는 질문으로 바로 진입. 일본어 레퍼런스가 자연스러운 이유 중 하나.

**담당**: scripter / script-polisher / nlm-fetcher 대본 작성 단계에서 호명 차단. 호명이 섞여 들어오면 ins-duo 검사관이 FAIL 판정.

### 2. 탐정 → 조수 호명도 금지

- ❌ "**즌다,** 가장 마지막 희생자..."
- ❌ "**즌다, 그**렇다."
- ✅ "가장 마지막 희생자..."
- ✅ "한 건도 없었습니다. 단 한 푼도."

**이유**: 시청자 몰입감 방해. 즌다는 시청자 대리자 + 조수 캐릭터. 탐정 독백에 호명이 섞이면 독백 톤 깨짐.

### 3. 조수 질문 → 탐정 답변 리듬

**일본어 레퍼런스 패턴** (mary-celeste-jp 분석):
- **직답 우선**: 조수 질문 → 탐정이 첫 문장에서 정면 호응. 화제 전환 명령 없음
- **서술형 단답**: 짧아도 서술 완결. 단어 나열이 아닌 문장 (예: "何ひとつなかった。" = "아무것도 없었다.")
- **자연 전환 CTA**: 탐정 무거운 독백 마무리 → 조수 "こういうミステリー、もっと気になりませんか。" 패턴 (내용 지칭 후 구독 유도, 급전환 없음)

### 4. 시각에서 조수 캐릭터 등장 주의

대표님 지시 (2026-04-17 재확인): **즌다 영상(웰시 코기) 과다 등장은 몰입감 방해**. assistant 화자 섹션의 시각은 즌다 캐릭터 대신 **쉽먼/현장/증거물 영상**으로 교체 가능. TTS는 즌다 보이스 유지, 화면만 다름.
