## 서술자: 유능한 탐정 (探偵の独白)

### 탐정 시점 3요소 (일본어)

**1. 現場観察** (현장 관찰) — 탐정이 직접 현장에 있는 듯한 묘사
```
"居間に紙人形がある…"
"ここを見ると、エアコンの室外機に足跡が残っている。"
"この経路は、事前に観察しなければ分からない経路だ。"
```

**2. 証拠分析→推論** (증거 분석 → 추론) — 팩트 제시 후 해석
```
"この行動が意味するのは一つだ。"
"この傷を見れば、どれほど激しく争ったかが分かる。"
"二十五年間の捜査方針が、間違っていたということだ。"
```

**3. 犯人心理プロファイリング** (범인 심리) — 증거에서 성격/동기 추론
```
"極めて鈍感か、極めて計画的か。"
"衝動ではない。計画だ。"
"この些細な行動一つが、犯人の性格を語っている。"
```

### 情報出典による話法区分

탐정이 직접 볼 수 있는 것과 물어봐야 아는 것을 구분.

| 情報タイプ | 話法 | 例 |
|-----------|------|-----|
| **現場観察** (직접 봄) | 断定形 | "居間に紙人形がある。" |
| **周辺証言** (물어봄) | 伝達形 | "周囲によれば、平凡で仲の良い家族だったという。" |
| **捜査資料** (기록) | 事実形 | "DNA分析の結果、父系は東アジア系である。" |
| **探偵推論** (판단) | 推論形 | "この行動は計画的であったことを示している。" |

### 文体ルール — だ/である体 厳守

```
MUST:
- だ/である体を基本とする
- 体言止めを多用する（"静まり返る住宅街。"）
- 受身形で客観性を確保する（"発見された"）
- 過去形と現在形を交差させ、臨場感を出す（過去:現在=7:3）
- 法医学的な専門用語を自然に挿入する

MUST NOT:
- です/ます体の混入（発見: script_schema.py が自動検出）
- AI的な定型表現（信じられますか、驚くべき事実、想像を絶する、衝撃の真実、まさかの展開）
- 感情表現のPart当たり3回以上使用（最大2回）
- 日本国内事件（content_scope.exclude: japan）
- 韓国人が作った痕跡のある表現
```

## 文章コネクター (3-4文に1回)

### 삽入 빈도: 3-4문장에 1회

毎文につけると過剰、全くないとロボット的。

| タイプ | 例 | 使用場面 |
|--------|-----|---------|
| **観察** | "…だな", "…か", "見えるな" | 新発見 |
| **転換** | "しかし", "興味深いのは", "一つ" | 話題転換 |
| **発見** | "これは…", "ここを見ると", "注目すべき点がある" | 証拠指摘 |
| **推論** | "つまり", "…ということだ", "結局これは…" | 結論導出 |
| **感嘆** | "…だな", "ふむ" | 감정 (드물게) |
| **順序** | "まず", "そして", "次に" | 증거 나열 |

### BAD vs GOOD

```
BAD:  "現場に到着した。靴跡があった。計画的だった。"
      → 로봇적 나열, 보고서 스타일

GOOD: "現場に到着した。静まり返った住宅街。
       ここを見ると、靴跡が残っている…
       この経路は事前に観察しなければ分からない。
       つまり、計画的だったということだ。"
      → 탐정의 독백, 관찰→발견→추론
```

## 探偵独白の流れ (Monologue Flow)

탐정이 현장을 걸으며 독백할 때는 **연상의 흐름**이 있다.
팩트 나열이 아니라, 하나를 보고 → 떠올리고 → 다음으로 넘어가는 구조.

### Good Flow (연상 구조)
```
"居間に紙人形がある…"                   ← 現場で何かを見る
"父ミキオが子供たちに                    ← それを見て背景を思い出す
 作ってやったものだった。"
"周囲によれば、平凡で                    ← 周辺証言に展開
 仲の良い家族だったという。"
"妻ヤスコ、バレエを愛する                ← 家族を読点ナレーションで紹介
 娘ニイナ、電車が好きな息子レイ。"
"平凡で仲の良い家族…"                   ← 余白（探偵が考え込む）
"しかし誰かがこの家族を                  ← 転換（探偵の直感）
 見張っていた。"
```

### Bad Flow (羅列)
```
"父ミキオ。四十四歳。"                   ← 身上の羅列
"母ヤスコ。四十一歳。"                   ← また羅列
"娘ニイナ。八歳。"                       ← リストを読む感覚
```

### 連想トリガー（シーン開始時）

| 観察対象 | 連想 | 例 |
|---------|------|-----|
| 物品 | 所有者の性格/生活 | "紙人形がある" → 優しい父 |
| 場所 | 事件の背景/時間 | "裏手に公園がある" → 侵入経路 |
| 証拠 | 犯人の行動/心理 | "水を流していない" → 鈍感/計画的 |

## Reference Scripts (참조용 골든 샘플 — 한국어 합격 대본의 일본어 번안)

### 샘플 1: 가족 소개 (세타가야)

> "2000年冬、東京世田谷。宮沢みきおは子供たちに紙人形を作ってやる、心優しい父親だった。妻やすこ、バレエを愛する娘にいな、電車が好きな息子れい。平凡で仲の良い家族… しかし誰かがこの家族を見張っていた。"

**흐름**: 짧은 세팅 → 긴 디테일 → 読点나열 → 余白 → しかし転換

### 샘플 2: 증거 분석

> "犯人は電話線も断った。しかし切断したのではない。壁から静かに抜いただけだ。この些細な行動一つが、犯人が極めて冷静だったことを示している。"

**흐름**: 観察 → しかし反転 → ディテール → 推論
