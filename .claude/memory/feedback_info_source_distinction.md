# feedback: 전달형 vs 직관형 구분

## 규칙

탐정이 직접 관찰한 정보 (현장 사진, 기록) 와 제3자 전달 정보 (경찰 발표, 주변 증언) 는 화법이 달라야 한다. 전달 정보를 직관처럼 단정형으로 쓰면 시청자가 정보 출처를 혼동한다 (incidents SKILL section 정보 출처에 따른 화법 구분).

## 예시 - 준수

직관: 현장 사진에는 피가 없었습니다. / 전달: 경찰은 피해자가 저항한 흔적이 없었다고 발표했습니다.

## 예시 - 위반

전달형 발표를 직관처럼: 피해자는 저항하지 않았습니다. (탐정이 그걸 어떻게 아는가? 혼동)
