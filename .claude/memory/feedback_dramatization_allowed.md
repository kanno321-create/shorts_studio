# feedback: 실화 기반 - 핵심 사실 엄격, 장면 디테일 각색 허용

## 규칙

incidents 채널은 실화 기반이지만 100% 논픽션 다큐가 아닌 Cinematic True Crime 다큐멘터리 포지션. 핵심 사실 (날짜/장소/피해자 수/판결 결과) 은 엄격히 사실에 근거하되, 현장 묘사 / 감각적 디테일 / 탐정의 호흡과 독백은 각색 허용. 단 허구 창작은 금지.

## 예시 - 준수

범행 시각 1999년 12월 30일 밤 11시 38분 (실제 기록 인용, 엄격) / 범인의 발자국 소리 복도에 울리는 낮은 마찰음 (장면 각색, 허용)

## 예시 - 위반

범행 시각 어느 겨울 밤 (엄격해야 할 사실 불분명) / 범인이 피해자에게 이렇게 속삭였다 (허구 대사 창작, 금지)
