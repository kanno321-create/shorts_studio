# feedback: 자막 6~12자, 동사 종결, 의미 단위 그룹핑

## 규칙

burned-in 자막은 6~12자 1줄로 의미 단위 그룹핑. 동사/서술어로 끝나야 하며 낱개 단어 쪼개기 금지. 의미 단위가 붙어있어야 시청자가 눈으로 훑으며 narration 과 동기화된다.

## 예시 - 준수

편지 1,701통이 배달됐습니다. (1줄, 의미 단위)

## 예시 - 위반

편지 / 1,701통이 / 배달됐 / 습니다 (4줄 낱개 분할)
